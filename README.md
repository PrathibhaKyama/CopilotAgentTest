# OctoFit Tracker

## Overview
The OctoFit Tracker is an application designed to help users track their fitness activities and progress.

## Features
- Activity tracking (running, cycling, swimming, etc.)
- Progress visualization (graphs and stats)
- Goal setting and management
- Notifications and reminders

## Installation
1. Clone the repository.
   ```bash
   git clone https://github.com/PrathibhaKyama/CopilotAgentTest.git
   ```
2. Navigate to the project directory.
   ```bash
   cd CopilotAgentTest
   ```
3. Install dependencies.
   ```bash
   npm install
   ```

## Usage
- To start the application, use:
   ```bash
   npm start
   ```
